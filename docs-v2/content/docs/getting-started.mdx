# Getting Started

This page demonstrates all standard Markdown/MDX typographic elements styled with the RetroUI neobrutalist design system.

## Inline Code

Use backticks to write `inline code` within a sentence. For example, call `bun install` to install dependencies, or reference a variable like `const config = {}`.

---

## Code Blocks Without Title

A plain code block — no title, copy button floats in the top-right corner.

```bash
bun install
bun run build
bun run dev
```

A TypeScript snippet without a title:

```typescript
export type Plugin = {
  name: string;
  version: string;
  tools: Tool[];
};
```

---

## Code Blocks With Title

A code block with a `title` attribute renders a RetroUI-styled title bar (yellow background, Archivo Black font) along the top.

```typescript title="platform/plugin-sdk/src/index.ts"
import { defineTool } from '@opentabs-dev/plugin-sdk';
import { z } from 'zod';

export const myTool = defineTool({
  name: 'my_tool',
  description: 'An example tool definition.',
  parameters: z.object({
    message: z.string().describe('The message to display'),
  }),
  async handle({ message }) {
    return { output: `You said: ${message}` };
  },
});
```

```bash title="Terminal"
# Start the MCP server in hot-reload mode
bun --hot platform/mcp-server/dist/index.js
```

```json title="~/.opentabs/config.json"
{
  "plugins": [
    "/Users/you/projects/my-plugin"
  ]
}
```

## Multi-Language Examples

**JavaScript**

```javascript
const greet = (name) => `Hello, ${name}!`;
console.log(greet('World'));
```

**Python**

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("World"))
```

**CSS**

```css
.card {
  border: 2px solid var(--border);
  box-shadow: 4px 4px 0 0 var(--border);
  background-color: var(--card);
  padding: 1rem;
}
```

**Shell**

```sh
# Install all workspace dependencies
bun install

# Build all platform packages
bun run build

# Run end-to-end tests
bun run test:e2e
```

## Lists

**Unordered list:**

- Plugin discovery via `opentabs-plugin.json`
- Tab state machine: `closed → unavailable → ready`
- WebSocket-based real-time communication

**Ordered list:**

1. Install the extension from `~/.opentabs/extension`
2. Start the MCP server with `bun --hot`
3. Add your plugin path to `~/.opentabs/config.json`
4. Reload the extension from `chrome://extensions/`
